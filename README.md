## 概要
androidスマートフォン向け「計測」アプリ　”　PhyPhox　”のcsvデータを簡易にグラフ化、マップ化できるソフトです。
今回は車両や徒歩でのルート追跡と速度に特化しています。

Location.csvとOrientation.csvを読み込んで、それらの数値を元にマップ上にルート表示できるようにしています。
（マップ表示にはHtmlとJavaを使用していますが、IEとか古いブラウザじゃない限り問題ないと思います。）
